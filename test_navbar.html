Modern Pill Navigation Test

You should see:

  • A light gray pill container around all navigation links
  • The "Packages" link with a golden gradient background
  • White text on the active "Packages" link
  • Hover effects on other links

If you DON'T see the pill navigation:

  1. Clear your browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
  2. Hard refresh the page (Ctrl+F5 or Cmd+Shift+R)
  3. Make sure you uploaded BOTH the style.css AND packageStyle.css files
  4. Check that the CSS files aren't being cached by your hosting
